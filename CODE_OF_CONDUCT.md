# Code of Conduct

## Our Commitment

We are committed to making participation in this project a respectful, harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive in all interactions.
- Focus feedback on technical content and project outcomes.
- Assume good intent, ask clarifying questions, and avoid personal attacks.
- Respect maintainers' time by providing reproducible details when reporting issues.

## Unacceptable Behavior

- Harassment, discrimination, hate speech, or abusive language.
- Doxxing, threats, intimidation, or sustained disruption.
- Publishing private information without explicit consent.
- Spam, trolling, or off-topic hostility in issues/PRs/discussions.

## Enforcement Responsibilities

Project maintainers are responsible for clarifying and enforcing this Code of Conduct. They may remove, edit, or reject contributions and interactions that violate these rules.

## Enforcement Guidelines

Maintainers may apply the following actions based on severity and pattern:

1. Warning and request to stop behavior.
2. Temporary restriction from discussions/issues/PRs.
3. Permanent ban from project spaces.

## Reporting

To report unacceptable behavior, open a private maintainer contact issue if available, or contact repository maintainers directly.

Include:

- Link(s) to relevant content
- Timeline and context
- Any supporting evidence

All reports will be reviewed promptly and handled as confidentially as possible.

## Scope

This Code of Conduct applies to project spaces including issues, pull requests, code reviews, and discussions, and also applies when representing the project in public spaces.
